# CHIP-8 Emulator

> I am working on this because I was interested in this topic. We'll see what I will learn from this. I will update this MD when I'm done with this project.

## Overview
This repository contains a personal project to build a **CHIP-8 Emulator**. CHIP-8 is an interpreted programming language from the 1970s, designed to allow video games to be easily programmed for early microcomputers. 

Writing an emulator is a significant milestone for any programmer interested in how computers work at a low level. This project serves as my introduction to the world of hardware emulation, opcode processing, and memory management.

## Why i am starting from CHIP-8?
CHIP-8 is the perfect starting point for emulation because:
- It has a small instruction set (only 35 opcodes).
- It uses a simple 64x32 monochrome display.
- It provides a clear understanding of the **Fetch-Decode-Execute** cycle.
